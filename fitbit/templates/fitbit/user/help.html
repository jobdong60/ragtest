자주 묻는 질문 (FAQ)
Q. Fitbit 계정은 어떻게 연동하나요?
'기기 등록 관리' 탭에서 'Fitbit 계정 연동하기' 버튼을 클릭하세요. Fitbit 로그인 페이지로 이동하며, 로그인 후 자동으로 데이터가 동기화됩니다.
Q. 데이터는 얼마나 자주 업데이트되나요?
자동 동기화는 5분마다 실행됩니다. 수동으로 동기화하려면 '데이터 동기화' 버튼을 클릭하세요.
Q. 어떤 데이터를 수집하나요?
다음 데이터를 수집합니다:
• 일일 활동 요약 (걸음 수, 거리, 칼로리)
• 안정 시 심박수
• 1분 단위 심박수 데이터
• 5분 단위 걸음 수 및 칼로리 데이터
Q. 데이터는 안전하게 보관되나요?
네, 모든 데이터는 암호화되어 안전한 PostgreSQL 데이터베이스에 저장됩니다. 개인정보 보호 정책에 따라 관리됩니다.
Q. Fitbit 기기가 없어도 사용할 수 있나요?
아니요, Fitbit 계정과 기기가 필요합니다. Fitbit 앱으로 수동 입력한 데이터도 동기화됩니다.

문제 해결
⚠ 데이터가 동기화되지 않아요
1. Fitbit 계정 연동 상태를 확인하세요.
2. '데이터 동기화' 버튼을 직접 클릭해보세요.
3. 로그아웃 후 다시 로그인해보세요.
❌ 로그인이 안 돼요
1. Fitbit 계정 아이디/비밀번호를 확인하세요.
2. 브라우저 쿠키 설정을 확인하세요.
3. 캐시를 삭제하고 다시 시도해보세요.
ℹ 데이터가 부정확해요
1. Fitbit 기기가 제대로 착용되어 있는지 확인하세요.
2. Fitbit 앱과 기기가 동기화되어 있는지 확인하세요.
3. 최소 24시간 후 데이터를 다시 확인해보세요.
